**COMPOUND INTEREST CALCULATOR CREATED USING PYTHON**

**Description:** 
A straight-forward and beginner friendly yearly compound interest calculator created using Python. Determines how much an investment balance grows over time using the Annual Compound Interest Formula (A = P * (1 + R/100)^t).

**Features:**

- User-friendly, easy to read logic.
- Input validation, checks for negative values and loops until valid user input is received using while loops, and if, else, and break statements. 
- Zero dependencies. No need to import or install external additional libraries.

**How to run:**
- Clone or download the repository
- Open the file using Visual Studio Code or a text editor of your choice.
- Run the program in your text editor's terminal.

**Requirements:**
- Visual Studio code or an equivalent modern text editor.
- Updated version of Python.
- No external libraries required.
